## ⚡ Get Started in 3 Steps

### Step 1: Preview Locally (5 minutes) — VS Code Only

> **Important:** All local previewing must be done through Visual Studio Code.

1. Open the project folder in **Visual Studio Code**
2. Install the **"Live Server"** extension (if not already installed)
3. Right-click `index.html` → **"Open with Live Server"**
4. Browser opens automatically at `http://127.0.0.1:5500`

### Step 2: Customize (30 minutes)

Edit `index.html`:
- Line 37: Hero title
- Line 58: Main description  
- Line 414: Your email
- Lines 510-520: Social links

Edit `css/style.css`:
- Lines 15-22: Change colors

### Step 3: Deploy (5 minutes)

1. **Sign up:** [netlify.com](https://netlify.com)
2. **Connect GitHub** (or drag folder)
3. **Add domain:** ciphertools.net
4. **Go live!** ✨
