Request a new secret key — verified by admin or moderator

🔒 Why do I need a new key? If your secret key has been compromised, leaked, or you suspect unauthorized access to your account, you should request a key reset immediately. A new key will be generated and your old key will stop working.

Secret Key Reset

How the key reset process works and what to expect.

🛡
Admin-Verified Process

Key resets are never automatic. An admin or moderator must review and approve every request to protect your account.

🔑
Password + Key = Your Code

Your reset code is generated from your password and secret key combined. This proves you own the account without exposing either credential.

🔄
Old Key Stops Working

Once approved, your old secret key is immediately invalidated. All apps must re-activate with the new key.

🔔
You’ll Get a Reset Code

After submitting, you’ll receive a unique reset code (RC-XXXXXXXX). Save it — it confirms your request was received.

🔒
Secure & Logged

All key reset requests are logged. Failed verifications produce no code and no request — attackers learn nothing.
