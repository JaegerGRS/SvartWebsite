Frequently Asked Questions

Is Svart Suite really free?

All tools except Svart AI are completely free with no limits. Svart AI gives you 750 free premium requests per month — the same full engine that paid users get, with no feature locks and no ads. If you need more, upgrade to £4.99/month for 2,000 requests or £9.99/month for unlimited. No ads on any plan, ever.

How does the activation key protect my privacy?

Each activation key generates a unique API key for your Local LLM. This API key is random and only becomes active after you log in to your app with your local password. Your activation key is your ID for the server only — the server cannot see or access your API key or your data. This ensures your encrypted AI is truly private and only you control it.

How is Svart Browser different from other browsers?

SVART is built specifically for privacy and security with military-grade encryption, zero extensions, built-in ad blocking, and Tor integration — all out of the box.

Can I install extensions?

In production mode (default), only our built-in ad blocker runs for maximum security. Dev mode allows extensions for developers and testers.

Will Svart Suite work offline?

Most features work offline. Ad blocking, encryption, and local features are always available. Svart AI and some cloud features require internet.

How secure is the master password?

Your master password is hashed with Argon2, a GPU-resistant algorithm. It's never stored in plain text and never sent anywhere.

What platforms are supported?

Windows 10/11, macOS 10.15+, and Linux (Arch, Ubuntu, Debian, Fedora). Native installers for all platforms.

How do I enable Tor?

Install Tor, then enable it in SVART settings. All traffic will route through Tor's network automatically.

What about my bookmarks and history?

All bookmarks, history, and credentials are encrypted locally. You control where they're stored.
