How It Works

Every account starts with 750 free AI chat requests per month. No card, no trial, no expiry. Requests reset on the 1st of each month.

Need more? £4.99/mo gives you 2,000 requests with £5/1,000 overflow. Want no limits? £9.99/mo gives you unlimited with priority processing.

All plans include the full local LLM engine, encrypted conversations, internet scraping, and custom training. The only differences are request volume and support priority.

What Counts as a Request?

One prompt + one AI response = one request

💬

Chat Messages

Each message you send to Svart AI and the response you receive counts as one request. Multi-turn conversations count each exchange.

🌐

Web Scraping Queries

When you ask Svart AI to search the internet for information, each scraping query counts as one request.

🤖

AI Training Prompts

Custom training prompts you feed to your local LLM to teach it new knowledge or behaviours each count as one request.

📄

Document Analysis

When you ask Svart AI to read, summarise, or analyse documents and files, each analysis counts as one request.
